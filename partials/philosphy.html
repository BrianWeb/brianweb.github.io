Design Philosphy

Mobile friendly characteristics include:

User friendly Layouts

Layouts are kept compact, with limited empty space to reduce scrolling / swiping.

Page elements are clearly defined and easy to see.

Minimalist, uncluttered design.

Single Page Design

Websites that can show new sections and information in a single web page (i.e. without the need extra new browser windows / tabs) This helps the web page appear and behave more like a mobile app.

Responsive Layouts

Multiple layouts to suit at least four different screen size ranges no matter what device the website is accessed from.

Responsive Images

To optimise load times, each image will have different size images. Which size image downloaded depends on the size of screens. Smaller size screens will download a smaller sized version, bigger screens – bigger versions, and medium size screens will be served a medium sized version of the same image.

This is achieved using HTML’s Responsive Images ‘srcset’ and ‘sizes’ attributes.

Accessibility

All web pages follow the W3C’s “Web Content Accessibility Guidelines 1.0" ( http://www.w3.org/TR/WCAG10-HTML-TECHS/) . This creates a more enjoyable web experience for anyone with accessibility issues. For example the content is made accessible to a user’s screen reader to help those who may have a sight impairment.
