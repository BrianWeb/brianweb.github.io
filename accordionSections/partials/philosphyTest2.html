User friendly Layouts

Layouts are kept compact, with limited empty space to reduce scrolling / swiping.

Page elements are clearly defined and easy to see.

Minimalist, uncluttered design.

Single Page Design

Websites that can show new sections and information in a single web page (i.e. without the need extra new browser windows / tabs) This helps the web page appear and behave more like a mobile app.

Responsive Layouts

Multiple layouts to suit at least four different screen size ranges no matter what device the website is accessed from.
