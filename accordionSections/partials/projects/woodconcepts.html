Wood Concepts

Wood Concepts is the largest Irish supplier of wood-based products to the Irish timber trade, with a supply chain covering South America, Asia, Europe and Russia.

The website was created with Angular 2x to make it as fast and responsive as possible. It uses the Boostrap framework, creating different layouts for different size screens (e.g., mobile/tablet / computer screens etc.) It’s content is updated using an API driven, ‘headless’ CMS, provided by Contentful.com.
