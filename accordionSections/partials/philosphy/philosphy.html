Tailor Made Designs

All websites are built using custom code, specific to your company's requirements. Sites are not confined to templates found on CMS packages such as Word Press or Drupal. This means if a design idea is technical possible to create (i.e. the technology is available), we can create it.

Another advantage of this approach is that the user only gets what they want. No bloated, load-time consuming, unnecessary code is left on their site.

Works On Any Device

Reponsive Design

Multiple layouts to suit any screen / viewport size, no matter what device the website is accessed from.


Single Page Design

Websites can show new sections and information in a single web page (i.e. without the need extra new browser windows / tabs). This helps the web page appear and behave more like a mobile app.

User Friendly Layouts

Layouts are kept compact, with limited empty space to reduce scrolling / swiping. Page elements are clearly defined and easy to see. Minimalist, uncluttered design.

Accessibility

All web pages follow the W3C’s “Web Content Accessibility Guidelines 1.0" ( http://www.w3.org/TR/WCAG10-HTML-TECHS/. This creates a more enjoyable web experience for anyone with accessibility issues. For example the content is made accessible to a user’s screen reader to help those who may have a sight impairment.

Speed

Responsive Images

To optimise load times, each image will have different sized versions. Which size image downloaded depends on the size of screens. Smaller size screens will download a smaller sized version, bigger screens – bigger versions, and medium size screens will be served a medium sized version of the same image.

Lightweight graphics

Vector graphics (SVGs) as icons (like the symbols on this page) mean smaller images and therefore faster load times.

AJAX

Ajax code allows for only part the of a webpage that is needed to be loaded.

Google Analytics

Websites can incorporate Google Analytics so that a business can gauge how its site is performing with regard to its content such as which links are clicked most often. It can also tell them or estimate the key demographic of its user base. These are just two examples of the many capabilities of Google Analytics.

For more, please see https://www.google.com/analytics/standard/features/

Search Engine Optimisation (SEO)

Website structure and content will developed with SEO in mind, putting the site as high up the search results as possible in Google, Bing, Yahoo and more. With the customer's consent, websites are continually checked (e.g. by keeping links and content up to date) to ensure SEO.
